# recipekr

시작 단계: Gemini 3.1 Pro를 사용하여 전체 MLOps/DevOps 파이프라인의 아키텍처 계획과 AWS 리소스 설계도를 먼저 그립니다.
구현 단계: Claude Sonnet 5를 코드 편집기(IDE)의 어시스턴트로 지정하여 Spring Boot 백엔드, Playwright 크롤러, Scikit-learn 모델 학습 스크립트를 세부적으로 작성하고 디버깅합니다.
CI/CD 및 컨테이너 단계: Claude Sonnet 5에게 Dockerfile과 .github/workflows/deploy.yml 작성을 요청하여 빌드 에러를 최소화합니다.
유지보수 및 고도화 단계: 전체 소스코드를 Gemini 3.1 Pro에 집어넣어, MLOps 관점에서 모델 자동 재학습 프로세스가 잘 돌아가는지 전체적인 코드 흐름을 최종 검토합니다.
